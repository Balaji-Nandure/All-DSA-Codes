Partitions with Given Difference

Split the array into two subsets S1 and S2 so that sum(S1) − sum(S2) = diff. That reduces to counting subsets with a fixed sum — classic 0/1 knapsack DP. Use tab 1 for the algebra, then tab 2 for the DP walkthrough.

Setup

DP

Array (processing order)
dp[j] = ways to form sum j
Index j (update) Index j − w Target column

In the 0/1 setting each array value is used at most once. The inner loop must run from high j to low j so that when you read dp[j − w], it still reflects the previous row (without the current element). If you loop forward, the same value can contribute multiple times in one pass — that becomes unbounded knapsack.

for (int j = target; j >= arr[i]; j--) // correct
  dp[j] += dp[j - arr[i]];

for (int j = arr[i]; j <= target; j++) // wrong for 0/1
  dp[j] += dp[j - arr[i]]; // reuses same element
